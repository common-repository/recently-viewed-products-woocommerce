== Description ==

Recently Viewed Products for Woocommerce is a WooCommerce addon plugin. It allows you to show the recently viewed products of the users on the product detail page and anywhere else on the website by using the predefined shortcodes.

= What The Plugin Does : =
Displays the most recently seen products on your Woocommerce store, sorted by user.

* on product page
* on shop page 
* on cart page

You can also use the [evdpl_rvps] shortcode to display the Recently Viewed Products. It's important to note that the shortcode uses the settings from the Settings Page. i.e.

* Should the slider be used or not?
* How many slides should be viewed, and so on.
* A slick slider can use to wrap products.
* If the user is signed in, it will add a record for that user id and be able to get all of the products the user has viewed.
* If the user is not signed in, a COOKIE is created to keep the ids of the browsed products.

= Benefits It Offers : =
* Display all user's recently viewed products on the product page.
* Set how many recently viewed products you want to display.
* Show a page with all recently viewed products of the user.
* Set the position of recently viewed products on the product page.
* Set title for recently viewed products section.
* Shortcodes to show Recently Viewed Products.
* Setting to display recently viewed products on the shop page or cart page.


== Installation ==
There are two methods by which you can install the plugin in your site. 

Method 1 (Upload via admin panel):
1. Go to WordPress admin then go to Plugins => Add new => Upload
2. Then select the installable zip file.
3. Upload and active the plugin. After that you can on setting page to adjust the view and title. It will start displaying on particular areas.

Method 2 (Upload via FTP):
1. Upload the plugin folder to your server plugin directory. Do not upload the zip file. It should be directly the folder.
2. After this, go to WordPress admin and go to Plugins => Installed Plugins
3. If the plugin is successfully uploaded then you will find "Woo Recently Viewed Products" here.
4. Active the plugin. After that you can on setting page to adjust the view and title. It will start displaying on particular areas.
